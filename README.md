# To-Do List

A minimal and elegant To-Do List web application. Manage your daily tasks efficiently with a clean interface and fast performance.

https://ayzsw.github.io/To-do-list/

---

## Features

- Add, edit, and remove tasks
- Mark tasks as completed
- Responsive user interface for mobile and desktop
- Local storage: tasks are saved in your browser
- Simple and elegant design
- No accounts or sign-in required

---

## Installation and Local Run

1. **Clone the repository:**
   ```bash
   git clone https://github.com/ayzsw/To-do-list.git
   cd To-do-list
   ```

2. **Open `index.html` in your web browser**

   No extra dependencies required.
